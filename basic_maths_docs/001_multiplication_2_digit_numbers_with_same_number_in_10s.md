# Two digit multiplication - Having same number in 10s position

## Step 1  - What are the two digit numbers?

![](001_two_2_digit_number.png)

## Step 2  - Example 1

1. Number 1: `13`

2. Number 2: `14`

```python
    result =  13 * 14
    result =  10 * ( 13 + 4 ) + (3 * 4)
    result =  10 *      17    +   12
    result =  170   +   12
    result =  182
```

## Step 3  - Example 2

1. Number 1: `23`

2. Number 2: `24`

```python
    result =  23 * 24
    result =  20 * ( 23 + 4 ) + (3 * 4)
    result =  20 *      27    +   12
    result =  540   +   12
    result =  552
```
